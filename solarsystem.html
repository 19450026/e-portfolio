phase description
Reconnaissance Gathering information about the target and identifying vulnerabilities.
Weaponization Crafting and preparing the exploit or malware.
Delivery Delivering the exploit to the target system.
Exploitation Executing the exploit to gain unauthorized access.
Installation Establishing persistence and creating backdoors.
Command and Control Taking control of the compromised system.
Actions on Objectives Performing the intended actions, such as data exfiltration or further attacks.

Mitigations for Each Phase:

phase Mitigation
Reconnaissance Implement network monitoring and intrusion detection systems to identify suspicious activities. Regularly update software and systems to patch vulnerabilities..
Weaponization Implement email security measures, such as spam filters and antivirus solutions, to detect and block malicious attachments or links.
Delivery Employ network filtering and firewalls to block known malicious domains or IP addresses. User awareness training to avoid clicking on suspicious links or opening attachments.
Exploitation Keep systems and software up to date with the latest patches and security fixes. Implement application whitelisting or sandboxing to limit the execution of unknown or unauthorized code.
Installation Implement strong access controls and least privilege principles to limit unauthorized access. Regularly monitor and analyze system logs for unusual activities.
Command and Control Use intrusion detection and prevention systems to identify and block malicious communications. Implement network segmentation to limit lateral movement
Actions on Objectives Implement data loss prevention measures to monitor and control data exfiltration attempts. Regularly backup critical data and verify the integrity of backups.

Tools for Each Phase:

phase Tools
Reconnaissance Shodan, Nmap, WHOIS, Google Dorks
Weaponization Metasploit, Cobalt Strike, Empire, Social Engineering Toolkit (SET)
Delivery Phishing frameworks (Gophish, King Phisher), Malware distribution platforms
Exploitation Exploit frameworks (Metasploit, ExploitDB), Web application scanners (Burp Suite, OWASP ZAP).
Installation Remote Access Trojans (RATs) such as Poison Ivy, Netcat, PowerShell Empire
Command and Control Command-and-control frameworks (C2Matrix, Covenant, Empire), DNS tunneling tools
Actions on Objectives Data exfiltration tools (PowerSploit, Exfiltration Over Websockets), Network scanning tools (Nmap, Nessus).
